# Motorola Sales Analysis Dashboard

## Project Overview
This project analyzes mobile phone sales data using Power BI.  
The dashboard provides insights into sales performance, customer behavior, and product demand.

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization

## Key Insights
- Total Sales: 769.0M+
- Total Transactions: 4K
- Sales by City
- Sales by Mobile Model
- Payment Method Analysis
- Customer Ratings


## Business Insights
- Apple and Samsung generated the highest revenue
- UPI and Debit Card are the most used payment methods
- Sales vary significantly across cities
